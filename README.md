## 🌐 Live Demo

Check out the live version of **Sawahel**:
👉 https://sawahel.vercel.app

**Sawahel** is a service-based platform that connects users with verified doctors and (soon) other services.
Users can browse doctors, manage their profiles, and securely authenticate using modern web technologies.

Built with a focus on:

* ⚡ Fast and responsive UI
* 🔐 Secure authentication (JWT + httpOnly cookies)
* 🧩 Scalable architecture (React + Zustand + Node.js)

More features and services are coming soon 🚀
